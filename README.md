
# React Native Basic Calculator 

## How to run this project

### Copy or clone this project
### cd project
### yarn install


## For Android to run this project

```bash
# using npx
npm run android

# OR using Yarn
yarn android
```

## For iOS to run this project

```bash
# using npx
npx react-native run-ios
```
